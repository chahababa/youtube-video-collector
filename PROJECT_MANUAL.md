# YouTube 影片收納器 — 使用說明書

## 這是什麼？

一個 Chrome 擴充功能，幫你把「等一下要看」的 YouTube 影片一鍵收起來，同時自動關閉分頁。不用再開一堆分頁佔記憶體，也不用去找 YouTube 內建的「稍後觀看」在哪裡。

---

## 安裝方式

1. 到 GitHub 下載專案：[youtube-video-collector](https://github.com/chahababa/youtube-video-collector)
   - 點綠色 **Code** 按鈕 → **Download ZIP**
   - 解壓縮到任意資料夾
2. 打開 Chrome，網址列輸入 `chrome://extensions`
3. 右上角開啟 **開發者模式**
4. 點 **載入未封裝項目**，選擇解壓縮後的 `youtube-video-collector` 資料夾
5. 工具列會出現外掛圖示（深灰色播放鍵＋箭頭），安裝完成

---

## 功能說明

### 1. 單部收納（左鍵點擊圖示）

在 YouTube 影片頁面，**左鍵點擊**工具列上的外掛圖示：

- 右上角會出現「已收納：{影片標題}」的提示
- 1.5 秒後該分頁自動關閉
- 影片資料已存入清單

### 2. 批次收納（右鍵選單）

開了一堆 YouTube 分頁？**右鍵點擊外掛圖示** → 選「收納此視窗所有 YouTube 影片」：

- 當前視窗內所有 YouTube 影片分頁會被收納並關閉
- **你正在看的分頁不會被關**
- 提示會告訴你收了幾部、跳過幾部重複

### 3. 查看已收納影片（側邊面板）

**右鍵點擊外掛圖示** → 選「已收納的影片槽」：

- 瀏覽器右側會打開面板，列出所有已收納影片
- 最新收納的排最上面
- 每筆顯示：影片標題、時長、收納日期

### 4. 面板內操作

| 操作 | 方式 |
|------|------|
| 播放影片 | 點擊藍色標題，會開新分頁播放 |
| 刪除影片 | 點該筆右邊的 ✕ 按鈕 |
| 搜尋影片 | 在頂部搜尋框輸入關鍵字，即時過濾 |

---

## 儲存說明

- 影片資料存在 **Chrome 帳號雲端同步**（chrome.storage.sync）
- 只要你有登入 Chrome，不同電腦之間會自動同步
- 最多可存約 **330 部影片**
- 到 **280 部**時會跳提醒，建議清理

---

## 哪些頁面可以收納？

| 頁面類型 | 能不能收 | 說明 |
|---------|---------|------|
| 一般影片 `youtube.com/watch?v=...` | ✅ | 正常收納 |
| 直播頁面 | ✅ | 時長會顯示「直播」 |
| YouTube Shorts | ❌ | 會提示「目前不支援短影片收納」 |
| YouTube 首頁 / 搜尋頁 | ❌ | 會提示「目前不在影片頁面」 |
| 非 YouTube 網站 | ❌ | 會提示「目前不在影片頁面」 |

## Q&A 常見問題

### Q：收納之後影片不見了？去哪裡找？
右鍵點擊工具列上的外掛圖示 → 選「已收納的影片槽」，側邊面板會列出所有已收納的影片。

### Q：同一部影片可以重複收納嗎？
不行，也不需要。如果你收納一部已經在清單裡的影片，會跳提示「這部已經在清單裡了」，而且分頁不會被關閉。

### Q：我換電腦了，之前收納的影片還在嗎？
只要你在兩台電腦都有登入同一個 Chrome 帳號，資料會自動同步。

### Q：影片清單有上限嗎？
有，大約 330 部。到 280 部時外掛會提醒你清理。建議定期把看完的或不想看的刪掉。

### Q：收納影片之後，那部影片如果被 YouTube 下架了怎麼辦？
清單裡的標題和連結還會在，但點進去播放時 YouTube 會顯示影片不存在。外掛只存連結，不存影片本身。

### Q：批次收納會把我正在看的那個分頁也關掉嗎？
不會。批次收納會跳過你目前的活動分頁，提示也會告訴你「已跳過你正在看的分頁」。

### Q：我用無痕模式可以用嗎？
預設不行，Chrome 無痕模式下擴充功能不會啟用。如果需要，可以到 `chrome://extensions` → 外掛的「詳細資訊」→ 開啟「在無痕模式中允許」。但無痕模式下不會同步資料。

### Q：怎麼更新外掛？
重新下載最新版的 ZIP，解壓縮覆蓋舊資料夾，然後到 `chrome://extensions` 點外掛的重新整理按鈕（↻）。已收納的影片資料不會消失。

### Q：怎麼完全移除外掛？
到 `chrome://extensions`，找到「YouTube 影片收納器」，點「移除」。移除後已收納的資料也會被清除。
